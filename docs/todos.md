# Todos — The Only One

## Ближайшие (MVP+)

- [ ] **Индикация загрузки на focus** — пока tasks грузятся из localStorage, показать скелетон/спиннер
- [ ] **Редактирование задачи** — свайп/тап на задаче открывает поле для уточнения
- [ ] **Отмена задачи** — кнопка "Пропустить" рядом с ГОТОВО, задача уходит в конец очереди
- [ ] **История дней** — хранить выполненные days в localStorage, показывать на главном экране
- [ ] **Авто-фокус поля ввода** — при загрузке `/` сразу ставить курсор в textarea

- [ ] **Офлайн-режим** — PWA + service worker с полным кэшем, фолбэк если нет API

- [ ] **Pull-to-refresh** на focus — перегенерировать план (с тем же контекстом)
